# EasyLaneRunner
EasyLaneRunner is a clean, modular, and event-driven framework designed to help Unity developers create lane-based runner games quickly and efficiently.

🚀 Key Features
Drag & Drop Ready: Easy to set up with your own prefabs.

Event-Driven Architecture: Decoupled logic using C# events for cleaner code.

Mobile Optimized: Built with performance in mind.

Modular System: Easily extendable for custom obstacles and power-ups.

🛠 Getting Started
Import the package into your Unity project.

Drag the LevelGenerator prefab into your scene.

Assign your obstacle and coin prefabs in the Inspector.

Hit Play and start running!

📂 Project Structure
Assets/NMDIOC.LaneRunner/: Main framework scripts.

Assets/NMDIOC.LaneRunner/Demo/: Includes a sample scene to get you started immediately.
